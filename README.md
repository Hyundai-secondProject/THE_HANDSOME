# HANDSOME

## 📌 Git Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[feat]**|새로운 기능 추가|
|**[fix]**|버그 수정|
|**[docs]**|문서 (문서 추가, 수정, 삭제)|
|**[test]**|테스트 코드|

ex) [Type] 커밋메세지 #이슈번호 <br>
[feat] 로그인 기능 구현 #12


## 📌 Git Branch
|브랜치명|작업 내용|
|:---|:---|
|**production**|배포|
|**main**|개발된 기능(feature)을 통합|
|**feat**|각 기능별 개발 진행|
